# FREDi

FREDi - **FR**emos **E**infacher **D**rehregler **I**nkremental (einige sagen auch **I**nnovativ :blush:)

Auch ich bin Besitzer einiger FREDis sowohl mit Impulsdrehgeber als auch mit Potentiometer.

Im Verzeichnis [Documentation](Documentation) sind Hinweise zu finden zu den Themen
- Upgrade eines FREDi mit 2.Shifttaste
- Bauanleitung eines FREDi mit Hardware V1.7 und 2.Shifttaster auf einer neu designten Platine
- Umbau eines FREDi mit Inkrementalgeber auf Potentiometer
- Kurzanleitungen zur FREDi-Bedienung
